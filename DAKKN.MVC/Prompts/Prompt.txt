Comprehensive Localization Verification & Fix Prompt (ASP.NET Core MVC)

First, read and analyze the entire repository before making any modifications. Do not fix files one by one without understanding the complete localization architecture.

Repository:
https://github.com/MohamedSaber2004/DAKKN

Objective

Perform a complete localization audit of the entire project to ensure that every user-facing text is displayed in the correct language (Arabic / English) according to the language requested by the current user.

This is not only about views.
